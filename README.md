# AirGuard VinUniversity AI Hackathon

Research and portfolio from the VinUniversity AI Hackathon, focusing on carbon emission analysis and AI-driven environmental monitoring.

## Contents

- **`carbon-portfolio.html`** — AirGuard carbon portfolio website showcasing the hackathon project
- **`dataset_research.html`** — Dataset research on CO2 emission factors across transportation, industry, and energy sectors

## About

**AirGuard** is an AI-powered carbon monitoring platform developed during the VinUniversity AI Hackathon. The project combines machine learning with environmental data to provide real-time carbon tracking and emission factor analysis.

## Tech Stack

Static HTML/CSS/JS — no build tools required.
